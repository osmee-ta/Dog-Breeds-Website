# 🐶 Dog Breeds Explorer

Dog Breeds Explorer is a responsive front-end web application built using **HTML**, **CSS**, and **JavaScript**. The project provides information about different dog breeds through an interactive interface featuring a rotating navigation menu, step-by-step navigation, a progress bar, and smooth animations.

## Features

* Step-by-step breed explorer
* Previous and Next navigation buttons
* Progress bar with step indicator
* Smooth transition animations
* Responsive design for desktop, tablet, and mobile devices
* Modern and user-friendly interface
* Keyboard navigation using Left and Right Arrow keys

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
